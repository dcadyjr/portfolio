Projects

Home Run Derby

This was my very first project of the General Assembly Web Development Immersive. In this project we were instructed to make a game. I love baseball, and a Home Run Derby game came quickly to my mind. In this game players have 10 outs to hit as many home runs as they can. I had a lot of fun adding little touches like sound effects for the bat and crowd, and text to taunt the players a bit.

Hashdash

This was a pair project in which we built a full stack application. We wanted a way to be able to follow hashtags we search frequently. We used the Twitter api to display tweets, and allowed users to save the hashtags they search frequently.
